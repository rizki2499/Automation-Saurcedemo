# Cara Menjalankan Skrip Automation Katalon
1.	Install katalon jika belum memiliki software
2.	Klik File -> Open Project
3.	Klik Test Cases -> Klik Test Case yang ingin di jalankan
4.	Klik tombol Running dan pilih chrome untuk menjalankannya
<img width="700" height="700" alt="image" src="https://github.com/user-attachments/assets/1756e3dd-b741-48ef-b86e-4dce284398e8" />


# Cara Memeriksa Laporan Uji Otomatisasi
1.	Pada saat klik tombol Running katalon otomatis menampilan page **"Job Progress"**
2.	Jika skrip yang dijalankan sesuai dengan hasil yang di harapkan dan tidak ada error. Maka, pada page **"Job Progress"** menunjukan hasill **"Passed"**
3.	Pada page bawah **"Log Viewer"** menunjukkan hasil **"Passed"** berwarna hijau
	<img width="700" height="700" alt="image" src="https://github.com/user-attachments/assets/040d5f41-6641-4836-bb00-70dc8c8bd905" />
